# Addinfo-Request

## Beispiel
https://openservice-test.vrr.de/openservice/XML_ADDINFO_REQUEST?filterShowLineList=0&filterShowStopList=0&outputFormat=rapidJSON&filterProviderCode=ZTP-PROD

## URL
https://openservice-test.vrr.de/openservice/XML_ADDINFO_REQUEST

Für weitere Informationen, siehe Beschreibung zur API unter:\
https://www.opendata-oepnv.de/ht/de/organisation/verkehrsverbuende/vrr/openvrr/api

## Mögliche Parameter
|Parameter|Werte|Beschreibung|
|-----------|-------|------------|
|filterShowLineList|0, 1|Aktiviert oder deaktiviert eine Ausgabe von Linien am Ende der API-Antwort| 
|filterShowStopList|0, 1|Aktiviert oder deaktiviert eine Ausgabe von Haltestellen am Ende der API-Anwort|
|outputFormat|rapidJSON, JSON|Format der API-Antwort|
|filterProviderCode|ZTP-PROD, ...|Filterung nach bestimmten Prodvider (ZTP-PROD für Meldungen von zuginfo.nrw)|
